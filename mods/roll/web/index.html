This application permits testing of the Saito Game Engine, and provides simply examples of working code that implements various core gameplay functions. These functions will work regardless of the number of players.

Some functionality like the generation of new (secure) dice rolls are not visible. Be sure to view the log for details on what is happening behind the scenes!
