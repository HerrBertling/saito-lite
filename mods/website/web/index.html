Welcome to the Saito Network

Most blockchains force users to pay for forever storage and require separate payments for off-chain data. Saito fixes both problems.

Saito applications only pay for the on-chain time they need, while the network uses a new kind of mining that secures the blockchain with fee-routing infrastructure. The result is lower fees and higher throughput than any other blockchain.

How Saito Works:

Saito is a layer-one blockchain similar to Bitcoin but with an improved incentive structure that eliminates the most common economic attacks on POW and POS blockchains. Those interested in learning how Saito works should see our simplified description of the network or jump to our more academic Saito Whitepaper. Key advances include:
Transaction Rebroadcasting

Saito prunes data by default, but allows old transactions to out-compete new ones for space on-chain. The result is a rent mechanism where prices are set by the market, not hardcoded by devs.

Transaction rebroadcasting eliminates blocksize limits. Maximizing profits requires block producers to target a fee-level which removes the same amount of data added each block. The blockchain can never collapse from over-use.

Saito Consensus

Saito uses a form of mining that pays the nodes which collect transactions from users. Cryptographic routing signatures and a provably-fair lottery make this as secure as mining.

Turning "efficient fee collection" into the network's form of work incentivizes the provision all routing infrastructure, software and data services needed to drive on-chain activity, incentivizing all activities needed to support essential network services.

About Saito

Saito is a solution to the problem of how to build a blockchain that preserves trustlessness at scale.

We prioritize building a real community with organic transactions and applications users want to use. The useability and usefulness of the network always takes priority. Saito is a powerful tool to build peer to peer applications, share messages and data securely, even transact other crypto.

Saito is an open network and users always own their data. We encourage anyone interested to get involved.
