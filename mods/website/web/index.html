Welcome to the Saito Network

Most blockchains force users to pay for forever storage and require separate payments for off-chain data. Saito fixes both problems.

Saito applications only pay for the on-chain time they need, while the network uses a new kind of mining that secures the blockchain with fee-routing infrastructure. The result is lower fees and higher throughput than any other blockchain.

How Saito Works:

Saito is a fundamental project most similar to Bitcoin, only without its economic problems and comcomitant attacks. Those interested in learning how Saito works can see our simplified description of the network or read the more academic Saito Whitepaper for a more substantive discussion of the underlying economic techniques. Key advances include:

Transaction Rebroadcasting

Saito prunes data by default, but allows old transactions to out-compete new ones for space on-chain. The result is a rent mechanism where prices are set by the market, not hardcoded by devs.

Transaction rebroadcasting eliminates blocksize limits. Maximizing profits requires block producers to target a fee-level where the amount of data removed each block matches the amount added. The blockchain can never get so bloated it collapses.

Saito Consensus

Saito uses a form of mining that pays the nodes which collect transactions from users. Cryptographic routing signatures and a provably-fair lottery make this as secure as mining.

Turning "efficient fee collection" into the network's form of work incentivizes the provision all routing infrastructure, software and data services needed to drive on-chain activity, incentivizing all activities needed to support essential network services.
